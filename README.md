# Eventra Web App

## Описание

Eventra - это веб-приложение для поиска клубов и организации мероприятий в вашем городе. Пользователи могут искать клубы, используя карту метро, а также изменять свои данные профиля и многое другое.

## Установка

1. Клонируйте репозиторий: `git clone https://github.com/your_username/eventra.git`
2. Перейдите в каталог проекта: `cd eventra`
3. Установите зависимости: `pip install -r requirements.txt`
4. Установите Yandex Maps API ключ, заменив `YOUR_API_KEY` в `index.html`.

## Использование

1. Запустите приложение: `python app.py`
2. Откройте браузер и перейдите по адресу `http://localhost:5000`

## Структура проекта

- `app.py`: Основной файл приложения.
- `static/`: Содержит статические ресурсы, такие как CSS, JavaScript, и изображения.
- `templates/`: Шаблоны HTML страниц.
- `constellation.js`: JavaScript файл для реализации звездного фона на главной странице.
- `main.js`: Основной JavaScript файл для логики приложения.
- `main_metro.css`: CSS файл для стилизации главного меню.
- `requirements.txt`: Файл с зависимостями Python.

# Система управления пользователями Flask

Это простая система управления пользователями, построенная с использованием Flask. Она включает функции регистрации пользователей, входа в систему, управления профилем и другие.

## Установка

1. Клонируйте репозиторий.
2. Установите зависимости с помощью `pip install -r requirements.txt`.
3. Запустите приложение Flask с помощью `python app.py`.
4. Перейдите по адресу `http://localhost:5001` в вашем веб-браузере.

## Особенности

- **Регистрация пользователя**: Пользователи могут зарегистрироваться с уникальным именем пользователя и паролем.
- **Вход пользователя**: Зарегистрированные пользователи могут безопасно войти в свои учетные записи.
- **Управление профилем**: Пользователи могут обновлять информацию в своем профиле, включая имя, телефон и электронную почту.
- **Загрузка файлов**: Пользователи могут загружать изображения профиля, которые хранятся безопасно.
- **Авторизация**: Некоторые маршруты защищены и требуют входа пользователя.
- **Регистрация ключа API**: Разработчики могут регистрироваться для получения ключей API для доступа к определенным конечным точкам.
- **Панель администратора**: Доступна панель администратора для выполнения административных задач.

## Структура файлов

- **app.py**: Основной файл приложения Flask с определениями маршрутов.
- **admin.py**: Шаблон для маршрутов, связанных с администрированием.
- **static/**: Каталог для статических файлов, таких как изображения и JSON-данные.
- **templates/**: Каталог для HTML-шаблонов.

## Зависимости

- Flask
- Flask-SQLAlchemy
- Werkzeug
- Python JSON
- SQLite3

## Использование

1. Зарегистрируйте учетную запись пользователя, используя конечную точку `/register`.
2. Войдите в систему, используя конечную точку `/login`.
3. Получите доступ к странице профиля, чтобы просмотреть и обновить информацию профиля.
4. Загрузите изображение профиля, используя конечную точку `/upload_image`.
5. Для разработчиков: зарегистрируйтесь для получения ключа API, используя конечную точку `/developer/register`.
6. Используйте ключ API для доступа к защищенным конечным точкам.

# Регистрация и Звездный фон

## Описание
Этот проект представляет собой страницу регистрации с использованием HTML и CSS для формы ввода данных пользователя, а также интерактивный звездный фон, созданный с использованием библиотеки Three.js.

## Установка
1. Склонируйте репозиторий: `git clone https://github.com/your_username/registration-star-background.git`
2. Откройте файл `index.html` в вашем веб-браузере.

## HTML и CSS
- `index.html`: Содержит HTML-разметку для страницы регистрации, а также включает ссылки на стили Bootstrap CSS и пользовательские стили из файла `style_reg.css`.
- `style_reg.css`: Файл со стилями для формы регистрации.

## JavaScript (Three.js)
- Скрипт Three.js находится внутри тега `<script type="module">` в конце файла `index.html`. Он создает трехмерный звездный фон с помощью библиотеки Three.js. Звезды двигаются по трем осям, создавая красочный эффект.

## Использование
1. Откройте файл `index.html` в вашем веб-браузере.
2. Заполните форму регистрации.
3. Нажмите кнопку "Зарегистрироваться", чтобы отправить данные.

## Дополнительная информация
- [Bootstrap](https://getbootstrap.com/): Используется для стилизации формы регистрации.
- [Three.js](https://threejs.org/): Библиотека для создания трехмерной графики в веб-браузере.

# Профиль пользователя

## Описание
Этот HTML-файл представляет собой страницу профиля пользователя, где пользователи могут просматривать и обновлять свою информацию. Он включает поля для имени, номера телефона и электронной почты, а также опцию загрузки изображения профиля.

## Особенности
- Отображение имени пользователя и его настоящего имени.
- Отображение изображения профиля пользователя (если доступно).
- Форма для обновления имени пользователя, номера телефона и электронной почты.
- Опция загрузки нового изображения профиля.

## Стилизация
- Страница имеет темный градиентный фон.
- Информация о пользователе и форма оформлены с закругленными углами и тенью.
- Изображение профиля отображается в круглой форме с рамкой.
- Поля ввода и кнопки имеют настраиваемый стиль с эффектами наведения.
- Макет реагирует на изменение размеров экрана.

## Использование
1. Заполните поля формы нужной информацией о профиле.
2. Нажмите кнопку "Сохранить изменения", чтобы обновить профиль.
3. Чтобы загрузить новое изображение профиля, нажмите кнопку "Загрузить" и выберите файл изображения.
4. Нажмите кнопку "Назад", чтобы вернуться на предыдущую страницу.

## Дополнительная информация
- Этот файл использует синтаксис шаблонов Jinja (`{{ ... }}`) для динамической вставки данных пользователя и URL-адресов.
- Функции JavaScript (`toggleProfileDrawer()`) используются для взаимодействия с выдвижным ящиком профиля.

<body>
    <div class="container">
        <h1>Изменение данных профиля</h1>
        <div class="user-info">
            <p><strong>Пользователь:</strong> {{ user.username }}</p>
            <p><strong>Имя:</strong> {{ user.name }}</p>
        </div>
        {% if image_path %}
            <img src="{{ url_for('uploaded_file', filename=image_path.split('/')[-1]) }}" alt="Изображение профиля">
        {% else %}
            <p>Изображение не загружено.</p>
        {% endif %}
        <form action="/update_profile" method="post">
            <input type="text" name="name" placeholder="Имя" required value="{{ user.name or '' }}">
            <input type="text" name="phone" placeholder="Номер телефона" value="{{ user.profile.phone or '' }}">
            <input type="email" name="email" placeholder="Email" value="{{ user.profile.email or '' }}">
            <input type="submit" value="Сохранить изменения">
        </form>
        <div id="profileDrawer" class="profile-drawer">
            <span class="close-btn" onclick="toggleProfileDrawer()">&#10005;</span>
            <img src="{{ url_for('uploaded_file', filename=image_path.split('/')[-1]) }}" alt="Профиль"
                style="width: 50px; height: 50px; border-radius: 50%;">
            <form id="uploadForm" action="/upload_image" method="post" enctype="multipart/form-data">
                <input type="file" name="profile_image" accept="image/*">
                <input type="submit" value="Загрузить">
            </form>
            <button onclick="window.location.href='/profile'">Изменить данные</button>
        </div>
        <!-- Кнопка Назад -->
        <a href="/" class="back-button">Назад</a>
    </div>
</body>
</html>
